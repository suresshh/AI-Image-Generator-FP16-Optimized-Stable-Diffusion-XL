System Architecture

1. Text Conditioning (CLIP Encoder)

User prompt is encoded into semantic embeddings using a frozen CLIP text encoder. These embeddings guide the denoising U-Net via cross-attention.

2. Forward Diffusion Process

Gaussian noise is gradually added to training images across timesteps, enabling the model to learn structured-to-noise transitions.

3. Reverse Diffusion (Denoising)

Starting from random Gaussian noise, the U-Net progressively removes noise to reconstruct meaningful visual structure.

4. Latent Space Representation

SDXL operates in compressed latent space using a VAE encoder, dramatically improving computational efficiency.

5. Text-Conditioned U-Net

Encoder-decoder architecture with skip connections and cross-attention layers inject prompt embeddings at multiple stages.

6. Current Model Architecture

The SDXL Base model generates global composition, lighting, and layout. It focuses on macro-level structure before refinement.

7. Refiner Model Enhancement

The Refiner model operates during later denoising steps to enhance textures, edges, shadows, and realism for high-resolution fidelity.

8. VAE Decoder Output

After denoising completes in latent space, the VAE decoder reconstructs the latent representation into a full-resolution image (e.g., 1024×1024).
